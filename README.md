# AI 简历筛查助手

一个基于 Deepseek AI 的智能简历筛查工具，可以批量分析候选人简历与职位需求的匹配度。

## 功能特点

- 📝 输入招聘需求（JD）
- 📄 批量上传候选人简历（最多10份PDF）
- 🤖 使用 Deepseek AI 智能分析匹配度
- 📊 生成详细的分析报告，包括：
  - 匹配度评分（0-100）
  - 候选人优势
  - 不足之处
  - 推荐建议（强烈推荐/推荐/待定/不推荐）

## 技术栈

- React 19 + TypeScript
- Vite 6
- Deepseek AI API
- PDF.js（PDF文本提取）
- Tailwind CSS 4
- Motion（动画效果）

## 使用说明

1. 在左侧输入招聘需求（职位描述、技能要求等）
2. 点击上传区域，选择候选人简历PDF文件（可批量选择，最多10份）
3. 点击"开始智能筛查"按钮
4. 等待AI分析完成，查看右侧的详细分析报告
5. 点击每个候选人卡片可展开查看详细信息

## 注意事项

- 仅支持PDF格式的简历文件
- 确保PDF文件可以正常提取文本内容
- API调用在前端进行，生产环境建议通过后端代理
